
# 2025_TODO_LISTS
新しい年を迎えたことを機に、新たな目標を設定し、実行に移すためにこのtodo listを作成いたしました。


+ 追加予定の機能
  1. Sign up後に他人の投稿も閲覧可能。各投稿にlikeやcommentも可能にする。
  2. Sign upを行わない場合は、自分の投稿のみ管理できる。
  3. 各投稿のstatusを自由に変更可能し視覚的に見易くする。
